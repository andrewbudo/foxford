### Ответ ###
Напишем следующие формулы:

A+x=2B+2y

A+x=2C+2z

B+y=C+z

x+y+z<=N

Решите их.

 

Если у системы есть решения, но одно из чисел отрицательное, либо сумма переменных больше N, то ответ 0.

Иначе выведите ответ.

 

a = int(input())

b = int(input())

c = int(input())

N = int(input())

y = (N - 3 * b + a + c - (N - 3 * b + a + c) % 4) // 4

x = 2 * (b + y) - a

z = b + y - c

if x < 0 or y < 0 or z < 0:

    print(0)

else:

    print(x, y, z)
